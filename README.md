小鱼方块 (Fish Puzzle)

复刻街霸方块

游戏所有素材由AI制作+我修改整合

竖屏横屏都可玩

移动：A / D (长按加速)
下落：S (长按加速)
旋转：J / K功能：左上角[暂停]，右上角[存档/读档]
完全免费，无广告核心玩法：
堆叠同色小鱼融合变大，消除鱼引爆连锁。

安卓苹果版有虚拟按键
双鱼对战为2台手机平板本地Wi-Fi联机
为了保证最佳体验，请留意：
1. 关于联机环境：部分旧型号手机的 WIFI 模块可能不支持点对点联机。
2. 对战重玩机制：由于联机对战中“重玩”功能依然存在不稳定性。
因此，我设定为一局分胜负。对战结束后，游戏将强制暂停15秒返回主菜单界面。如需再战，请重新连接。

安卓版如提示未知应用在手机设置允许安装如下载后缀名apk.1用自带浏览器安装
